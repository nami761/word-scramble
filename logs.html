フォームズのログを反映させる方法
  1. フォームズ（formzu.com） にログインしてフォームを開く
  2. 「受信データ」→「CSVダウンロード」でデータを取得
  3. ダウンロードした CSV ファイルの名前を logs.csv に変更
  4. GitHub のこのリポジトリを開き、logs.csv ファイルを選択 → 鉛筆アイコン（編集）→ 内容を貼り付けて「Commit changes」
  5. 数分後に GitHub Pages が自動更新され、このページに反映されます
※ CSV の1行目はヘッダー行として扱われます。フォームのフィールド名がそのまま列名になります。
