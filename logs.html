このページを最新データに更新する方法
  1. フォームズ（formzu.com） にログインして対象フォームを開く
  2. 「受信データ」→「CSVダウンロード」をクリック
  3. ダウンロードしたファイル名を logs.csv に変更する
  4. GitHub のリポジトリで logs.csv を開き、鉛筆アイコン（Edit）をクリック
  5. 内容を全選択して削除 → ダウンロードした CSV の内容を貼り付けて「Commit changes」
  6. 数分後にこのページが自動更新されます
※ メールで届いたログをコピーして手動で行を追加することもできます。
